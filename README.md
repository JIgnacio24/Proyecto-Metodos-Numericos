# Proyecto de Métodos Numéricos

Proyecto desarrollado en **MATLAB** para aplicar y comparar distintos métodos numéricos clásicos en la aproximación de raíces, derivación numérica e interpolación.

## Descripción

El programa presenta un menú interactivo que permite ejecutar diferentes algoritmos numéricos orientados a resolver problemas matemáticos de aproximación y análisis.

## Métodos implementados

### 1. Comparación de métodos para aproximación de raíces
- Bisección
- Newton-Raphson
- Secante
- Punto fijo

### 2. Derivación numérica

### 3. Interpolación de Lagrange
